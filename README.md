# ⚡ WuWa Mod Woju Fixer (WWMI)
A simple tool to fix broken Wuthering Waves mods after game updates.
If you’ve ever opened the game after a patch and half your mods stopped working… yeah, this is for that.

---

## 🚀 What it does
- Fixes mods made using WWMI  
- Browse & Fix (Simple & User Friendly)
- Can add stable textures to avoid texture glitch when in open-world
---

## 📥 How to install
1. Go to the releases page  
2. Download the latest version  
3. Extract zip
That’s it.

---

## 🛠️ How to use
1. Open the app
2. Select your mod folder
3. Enable StableTextures Fix if you need it
4. Click **Run Fix**  
5. Wait a bit  
6. Done 👍  

Your fixed mods will be saved automatically.

---

## ⚠️ Before you ask
- ❌ This doesn’t magically fix *everything*  
- ❌ It won’t support future patches instantly  
- ⚠️ Can only fix character mods right now, for weapons etc might still need to wait for future update
