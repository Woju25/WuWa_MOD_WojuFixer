# ⚡ WuWa Mod Woju Fixer (WWMI)
A simple tool to fix broken **Wuthering Waves** mods after game updates.  
If you’ve ever opened the game after a patch and half your mods stopped working… yeah, this is for that.

---

## ⬇️ Download (Recommended)
**Do NOT download from the green “Code” button.**  
Go to **Releases** and download the latest `.zip` from there.

After downloading:
1. Extract the `.zip`
2. Run `WuWa_MOD_WojuFixer.exe`

---

## ✅ Requirements / Important notes (Read this)
To avoid confusion, this tool has a few requirements:

- **You must have the latest RabbitFX installed** (and set up correctly) for the mods you’re trying to fix.  
  If RabbitFX itself is outdated, some mods can still look broken even after this tool updates `mod.ini`.

- This tool is intended for **WWMI character mods** (not everything).
- The app modifies `mod.ini` files but **creates backups first**.
- Might not work for fixing mods with custom mod.ini naming and/or custom textures

---

## 🚀 What it does
- Fixes character mods made using **WWMI**
- Simple “Browse → Run Fix” workflow
- Optional **StableTextures** injection to help prevent texture glitches in open-world
- Creates backups before applying changes

---

## 🛠️ How to use
For visual learner, watch my video instead --> 
1. Open the app
2. Select your mod folder (the folder that contains the `mod.ini`)
3. (Optional) Enable **StableTextures Fix** if you need it
4. Click **Run Fix**
5. Wait a bit
6. Done

Your fixed mods will be saved automatically (backups are created).

## ⚠️ Before you ask
- ❌ This doesn’t magically fix *everything*
- ❌ It won’t support future patches instantly
- ⚠️ Currently focused on **character mods** (weapons/other assets may need future updates)
- ⚠️ If a mod is very old / heavily edited, it may still need manual fixing
- ⚠️ This app is intended only for temporary textures fix, when you zoomed-out the camera too far, mod will still disappear (need to wait for XXMI update)
